This repository contains a tool I made that takes 2 inputs:
  
  ##### parameters
  _numberofAttributes_: an integer
  _supportThreshold_: a float
  
  ##### constraints
  1<= _numberofAttributes_ <= # of features
  0<= _supportThreshold_ <= 1.0
